# 条件运算符

条件运算符，也叫三元运算符。

## 语法

条件表达式?语句1:语句2;

## 流程

条件运算符在执行时，先对条件表达式进行求值。

如果该值为true，则执行语句1，并返回执行结果。

如果该值为false，则执行语句2，并返回执行结果。

```js
var a = 10;
var b = 20;
a>=b?console.log("a>=b"):console.log("b>a"); //输出b>a
var max = a>=b?a:b; //max=20
```

*欠打的取三个数的最大值的写法*

```js
var max = a>b?a>c?a:c:b>c?b:c;
//var max = a>b?(a>c?a:c):(b>c?b:c);
```

不推荐使用，不方便阅读。



**如果条件表达式的求值结果是一个非布尔值，会先转换成布尔值，再进行判断。**
