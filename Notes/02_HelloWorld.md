# HelloWorld

JavaScript代码需要写到`<script>`标签中

`<script type="text/javascript"></script>`



`alert()`:控制浏览器弹出警告框



```html
<script type="text/javascript">
	alert("Hello World!");    
</script>
```



`document`网页文档

`document.write()`:向`<body>`中写一个内容



`console`控制台

`console.log()`:向控制台输出一个内容



+ JS代码从上到下顺序执行
