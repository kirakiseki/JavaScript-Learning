# 其他进制的数字

表示其他进制的数字时，需要加上修饰符。在输出时统一以十进制输出。

| 进制                     | 修饰符 |
| ------------------------ | ------ |
| 十六进制                 | 0x     |
| 八进制                   | 0      |
| 二进制（部分浏览器支持） | 0b     |

```js
var a = "070";
a = parseInt(a);
console.log(a);
```

* 如"070"这种字符串，有些浏览器会当成8进制解析，而有些会当成10进制解析。

  * 可以在parseInt()中传递第二个参数，来指定数字的进制。

    ```js
    a = parseInt(a,10); //指定将a作为10进制数解析
    ```
